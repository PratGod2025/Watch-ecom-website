# Watch Website

A website project for watches.

## Description

This is a watch e-commerce or information website built with HTML, CSS, and JavaScript.

## Files

- `index.php` - Main PHP file

## Installation

1. Clone the repository inside htdocs of xampp
2. Open `http://localhost/Time-Hub/Watch-ecom-website/index.php` in a web browser

## Usage

Navigate through the website to view watch products and information.
